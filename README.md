# 🌤️ SkyFetch - Weather Dashboard

SkyFetch is a weather dashboard that allows users to search for any city and view real-time weather information along with a 5-day forecast.

The application uses the OpenWeatherMap API to fetch weather data and displays it in a clean and responsive interface.

---

## 🚀 Live Demo

Live Website:  
https://skyfetch-weather-dashboard-beta.vercel.app/

---

## ✨ Features

- Search weather for any city in the world
- Display current temperature and weather conditions
- Show weather icons
- 5-Day weather forecast
- Recent searches saved using localStorage
- Auto-load last searched city
- Responsive design for different screen sizes

## 💻 Local Setup

1. Clone the repository git clone https://github.com/chiranthanreddy-cpu/skyfetch-weather-dashboard.git

2. Open the project folder.

3. Open **index.html** in your browser.

4. Replace the API key in **app.js** with your own OpenWeatherMap API key.

---

## 🔑 API Used

OpenWeatherMap API  
https://openweathermap.org/api

- Current Weather API
